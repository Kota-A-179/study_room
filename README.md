# アプリケーション名

Study Room

# アプリケーション概要

共用の自主学習スペースで人が感じる刺激を再現し、「自宅で人を感じながら勉強できる」を実現します。

# 利用方法

## 学習開始

1. ログイン後、フッターに表示されている「学習開始」から学習タイトルを入力して新規登録します。
2. 必要に応じて「記録する」から、目標や学習内容や振り返りについて入力し「更新する」で保存します。
3. 取り組んでいる内容を終了する場合は「記録する」からページ遷移し、下部にある「完了する」で内容を保存します。
4. 別の学習を取り組む場合は再度「学習開始」から始めます。学習を終える場合はヘッダーの「退室する」でログアウトします。

## ユーザー情報・学習記録の確認

・ヘッダーの「マイページ」からユーザー情報や記録した学習内容・学習時間が確認できます。自分のマイページであればユーザー情報や学習内容について編集できます。
・ヘッダーの「Study Room」からルームの詳細画面に戻ることができます。  
・ルームの詳細画面で表示されているアイコンから各ユーザーのマイページに遷移し、ユーザー情報や学習記録を閲覧できます。

# 開発背景

開発の背景は、自宅での自主学習が思うように進まなかったことにあります。自主学習に取り組みたい気持ちはありましたが、学習に身が入らず中々進めることができませんでした。
図書館などの共用の学習スペースでは取り組むことはできますが、「行き返りに時間が掛かる」「昼食等の準備が必要」「スペースが狭い」などの問題があり、可能であれば自宅で学習したいと考えていました。  
そこで自宅で学習することが困難な原因を探り、原因はモチベーションの低さではなく自宅では刺激がないことにあると考えました。  
そのため自主学習スペースで感じる刺激を再現し、「自宅で人を感じながら勉強できる」を実現するアプリケーションを開発することにしました。  
また、自主学習に関するユーザーが抱えている課題として、「学習塾に通いたくても金銭や距離の問題で困難」ということも考えられました。その課題に対するアプローチとしても、「自宅で人を感じながら勉強できる」の実現は有効と思われ、より開発の意義を感じました。

# 工夫したポイント

## 学習に集中しやすいこと

・ルームとユーザーの自動紐づけや、学習時間の自動計測などで UI をシンプルにしています。
